## Overview

Base path: `/api/v1`. Send `Authorization: Bearer <access_token>` for protected routes.

### Login

POST `/api/v1/auth/login`

Request
```json
{
  "email": "user@example.com",
  "password": "strongpassword"
}
```

Response
```json
{
  "success": true,
  "access_token": "<jwt>",
  "refresh_token": "<jwt>",
  "session_id": "...",
  "expires_at": 1735689600,
  "user": {
    "id": "...",
    "email": "user@example.com",
    "first_name": "...",
    "last_name": "...",
    "role": "student",
    "is_active": true,
    "email_verified": true,
    "created_at": 1731000000
  }
}
```

### Register

POST `/api/v1/auth/register`

Request
```json
{
  "email": "new@example.com",
  "password": "Password123!",
  "first_name": "New",
  "last_name": "User",
  "role": "student",
  "organization": "",
  "phone_number": ""
}
```

### Validate Token

POST `/api/v1/auth/validate`

Request
```json
{ "token": "<jwt>" }
```

Response
```json
{ "valid": true, "user_id": "...", "role": "student", "expires_at": 1735689600, "session_id": "..." }
```

### Me

GET `/api/v1/auth/me` (auth)

Returns current user profile.

### Check Permission

GET `/api/v1/auth/check-permission?resource=users&action=read` (auth)

Response
```json
{ "allowed": true, "resource": "users", "action": "read", "user_id": "..." }
```

### Logout

POST `/api/v1/auth/logout` (auth)

Request
```json
{ "session_token": "...", "access_token": "<jwt>", "refresh_token": "<jwt>" }
```
