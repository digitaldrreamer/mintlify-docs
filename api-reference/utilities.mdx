Public endpoints (no auth):

- GET `/health` — health status
- GET `/ready` — readiness
- GET `/live` — liveness
- Docs UI: `/docs`, plus `/docs/openapi.yaml`, `/docs/swagger.yaml`, `/docs/redoc`, `/docs/swagger`, `/docs/auto`, `/docs/swagger.json`
