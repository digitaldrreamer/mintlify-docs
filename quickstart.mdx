## Authenticate

Use the login endpoint to obtain a JWT and session ID.

Request
```bash
curl -X POST "$BASE/api/v1/auth/login" \
  -H "Content-Type: application/json" \
  -d '{"email":"user@example.com","password":"Password123!"}'
```

Response (truncated)
```json
{ "access_token": "<jwt>", "refresh_token": "<jwt>", "session_id": "..." }
```

## Call an endpoint

Pass the Bearer token in the Authorization header.

```bash
curl "$BASE/api/v1/users?limit=10" \
  -H "Authorization: Bearer $TOKEN"
```

## Upload biometric image (multipart)

```bash
curl -X POST "$BASE/api/v1/biometric/register" \
  -H "Authorization: Bearer $TOKEN" \
  -F "image_data=@face.jpg" \
  -F "type=0"
```

## Try it in the browser

Open the API Reference and click Authorize to paste your token.
